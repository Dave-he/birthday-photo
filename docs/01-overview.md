# 1. 项目概览 (Project Overview)

## 1.1 项目愿景
本项目旨在打造一款温暖、梦幻且易于管理的 3D 电子相册。区别于传统的二维图片轮播，它通过 3D 互动的形式（如旋转的圣诞树、漂浮的螺旋画廊），将珍贵的回忆悬挂其中。用户可以在指尖旋转、探索，感受每一张照片背后的故事。

它不仅仅是一个照片展示工具，更是一份充满心意的数字化礼物，特别适合作为生日礼物、圣诞惊喜或纪念日留念。

## 1.2 核心价值
- **情感连接**：通过沉浸式的视觉体验和背景音乐，唤起用户对美好回忆的共鸣。
- **沉浸式体验**：利用 WebGL 技术构建 3D 场景，配合动态天气（下雪、烟花）和后期特效，营造身临其境的氛围。
- **易于管理**：配套的后台管理系统让非技术人员也能轻松上传照片、撰写祝福语并配置场景。

## 1.3 核心特性

### 1.3.1 多场景模式 (Multi-Scene Modes)
系统内置多种主题模式，可根据场合自动或手动切换：
- **🎄 Christmas (圣诞模式)**：核心场景为一棵装饰着照片球的 3D 圣诞树，伴随飘雪效果。
- **🎂 Birthday (生日模式)**：场景包含生日蛋糕、漂浮气球，色调明亮欢快。
- **💖 Romantic (浪漫模式)**：粉色调氛围，包含爱心元素和柔和的粒子效果。
- **🎉 Party (派对模式)**：动感的烟花特效，适合庆祝场合。

### 1.3.2 互动体验
- **3D 漫游**：用户支持触摸或鼠标拖拽，360度旋转观察场景。
- **点击交互**：点击场景中的照片，相机自动推近聚焦，并弹出高清大图及祝福语卡片。
- **HUD 控制**：屏幕边缘提供悬浮控制栏，可切换场景、模式、布局及控制音乐。

### 1.3.3 个性化定制
- **全局配置**：管理员可设置背景音乐、欢迎语标题、是否下雪等。
- **多布局切换**：照片展示支持树形 (Tree)、螺旋 (Helix)、球形 (Sphere)、网格 (Grid) 等多种空间排列方式。

## 1.4 适用人群
- **送礼者**：希望为亲朋好友制作独特电子礼物的用户。
- **活动策划**：需要在派对、婚礼等现场大屏展示照片墙的组织者。
- **开发者**：希望学习 Next.js + React Three Fiber 结合开发的工程师。
